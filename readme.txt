authorisation - экран авторизации пользователей в систему, при вводе некорректных данных пользователем выдается сообщение об ошибке.
card_screen - интерфейс редактирования, сохранения, распечатки ранее созданных или создаваемых в данный момент учетных документов.
main_screen - главный экран программы, содержащий фильтры для производства выборки зданий, интерфейсы создания докуметов, редактирования справочников и администрирования.
handbooks - справлочный интерфейс отображающий рабочую информацию, необходимую пользователям для создания документов.
